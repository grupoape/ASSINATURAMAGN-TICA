V66 - REAÇÕES GLOBAIS EM TEMPO REAL

Esta versão corrige curtidas/deslikes/favoritos com totais diferentes entre contas.

IMPORTANTE:
Para funcionar entre computadores e redes diferentes, o Firestore Database precisa estar ativo no Firebase.

PASSOS:
1. Acesse https://console.firebase.google.com/
2. Abra o projeto assinatura-magnetica.
3. Vá em Firestore Database.
4. Crie o banco se ainda não existir.
5. Para teste, use Start in test mode.
6. Publique novamente os arquivos no GitHub Pages.

DOCUMENTO USADO:
assinaturaMagnetica/promptReactionsRealtime

Sem Firestore ativo, as reações continuam locais por navegador.
